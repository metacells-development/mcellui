## Overview

mcellui includes 43 production-ready components designed for Expo and React Native. Each component:

- Works on iOS and Android
- Supports dark mode
- Is fully accessible
- Includes TypeScript types
- Uses Reanimated for animations

## Quick Add

Add all components at once:

```bash
npx mcellui add button card input dialog sheet toast tabs
```

Or use the interactive picker:

```bash
npx mcellui pick
```

## Component Status

All components are in **beta** status. This means:
- API is stable but may have minor changes
- Tested on iOS 17+ and Android 13+
- Production-ready for most use cases
